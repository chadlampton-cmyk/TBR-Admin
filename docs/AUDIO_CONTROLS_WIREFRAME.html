Why This Is Different
  • `Music Master` is the left control strip, not another generic card.
  • `Active Cues` is now a channel rack, which reads more like a real mixer.
  • `Voice` and `Output` are utility panels, not equal-weight clutter.
Engine Direction
  • Each cue row maps to its own Web Audio channel.
  • The left strip controls the master music bus.
  • The right stack stays lightweight so the browser is not doing unnecessary UI work.
